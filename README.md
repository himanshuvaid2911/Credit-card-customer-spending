# Credit Card Customer Spending Analysis & Segmentation

## Overview
This project analyzes credit card transaction data to understand customer spending behavior and segment customers based on their transaction patterns.

## Dataset
- Credit card transactions across Indian cities
- 26,000+ transaction records

## Tools & Technologies
- Python (Pandas, NumPy, Scikit-learn)
- Power BI

## Key Steps
- Data cleaning and preprocessing
- Feature engineering
- Customer segmentation
- Interactive dashboard creation

## Insights
- High-value customers contribute the majority of total spend
- Metro cities dominate credit card usage
- Bills and groceries are the largest expense categories
- Monthly spending trends show seasonal patterns

## Dashboard
Power BI dashboard includes KPIs, slicers, customer segmentation, city-wise analysis, expense analysis, and monthly trends.
